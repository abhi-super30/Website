## 🚀 Deployment

1. **Clone Repo**
   ```bash
   git clone https://github.com/<your-username>/the-codecrush.git
   cd the-codecrush
````

2. **Install Dependencies**

   ```bash
   npm install
   ```

3. **Run Locally**

   ```bash
   npm start
   ```

4. **Build for Production**

   ```bash
   npm run build
   ```

5. **Deploy**

   * GitHub Pages → simple static hosting
   * Netlify → drag and drop build folder
